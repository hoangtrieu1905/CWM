# CWM
Web-based mini-game (CWM) based on the BrainScale.net cognitive training exercise.
Source: https://brainscale.net/app/brain-training/complex-working-memory/tutorial 

This project is a non-commercial, educational replication. Its sole purpose is to practice and demonstrate software development principles by recreating the cognitive training mechanics outlined in the Complex Working Memory (CWM) Tutorial.

Original Concept: All core game mechanics, including the Spatial CWM and Verbal CWM variations, are directly modeled after the exercises provided by BrainScale.net. I claim no ownership over the original game design, concepts, or rules.
